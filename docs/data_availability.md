# Data Availability

The staged monthly wind-generation series is provided as a simple CSV for reproducible demonstration purposes.

It is a public-safe derived dataset extracted from the coursework material and limited to:

- `date`
- `wind_generation_mw`

The original Excel workbook from the course folder is not distributed here.

Additional small text files included in `examples/` are compact univariate practice series and are kept only as simple forecasting examples.
